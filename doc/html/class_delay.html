Detailed Description

template<unsigned int NUM_CELLS>
class Delay< NUM_CELLS >

A delay line, with a variable delay up to NUM_CELLS long, NUM_CELLS should be a power of two.

The largest delay you'll fit in an atmega328 will be 512 cells, which at 16384 Hz sample rate is 31 milliseconds. More of a flanger or a doubler than an echo. This version doesn't have feedback, for that use DelayFeedback instead.
