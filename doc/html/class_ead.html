Detailed Description

Exponential attack decay envelope.

This produces a natural sounding envelope. It calculates a new value each time next() is called, which can be used to change the amplitude or timbre of a sound. The equations used are: attack: y = a*pow((1+r),x) decay: y = a*pow((1-r),x)
