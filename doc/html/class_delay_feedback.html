Detailed Description

template<unsigned int NUM_TABLE_CELLS>
class DelayFeedback< NUM_TABLE_CELLS >

A delay line, with a variable delay up to NUM_TABLE_CELLS long and feedback, NUM_TABLE_CELLS should be a power of two.

The delay length is half that of a plain Delay object, in this case 256 cells, or about 15 milliseconds. DelayFeedback uses ints to accomodate the direct input to the delay as well as the feedback, without losing precision. Output is only the delay line signal. If you want to mix the delay with the input, do it in your sketch.
